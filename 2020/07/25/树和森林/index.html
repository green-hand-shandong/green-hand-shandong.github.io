树和森林

00 考点

  • 树的存储结构，树、森林和二叉树的相互转换、树和森林的遍历运算和基本运算
  • 高频：树、森林、二叉树的相互转换
  • 主要客观题，算法少

01 知识

  • 了解，双亲表示法：两个域的数组，data、parent。

    • 方便找结点双亲、查找孩子难、顺序存储结构

  • 了解，孩子表示法

    • 结点的度，设置结点的指针个数
    • data、child1、child2、child3、child4
    • 节省存储。不同的数据元素节点构造不一样。操作不便

  • 了解，孩子表示法

    • 按照树的度进行设置指针个数

  • 了解，孩子链表法

    • 顺序（结点） + 链表（孩子）
    • 找孩子容易，找双亲难

  • 了解，双亲孩子链表法

    • 顺序（结点+parent）+链表（孩子）

  • 重点，左孩子右兄弟表示法 / 二叉树表示法

    • firstchild、data、nextSibiling(下一个兄弟)
    • 树转二叉树

  • 树定义

  • 树到二叉树转换

    • （手工模拟）
      • 连线，在兄弟结点之间连线，
      • 删线，每个结点除了保留与其最左孩子的连线外，删去与其他孩子之间的连线
      • 旋转，顺时针旋转 45°
    • 根节点右子树一定为空，根节点没有右兄弟

  • 森林到二叉树转换

    • （手工模拟）
      • 连线，在兄弟结点之间连线。包括森林中所有树的根节点。
      • 删线，每个结点除了保留与其最左孩子的连线外，删去与其他孩子之间的连线
      • 旋转，顺时针旋转 45°
    • 所有子树的根节点都是兄弟
    • 根节点的右子树上有多少右子树根节点，森林就有多少子树。

  • 二叉树到森林

    • 连线，双亲–孩子，先连成虚线
    • 删线，删除兄弟之间的连线
      • （节点的右指针指向孩子），删除的是根节点–右指针
    • 逆时针旋转45°，虚线转实线

  • 重点：树和森林的遍历

    • DFS：前序、后序
    • BFS：层次遍历

  • 前序

    • 非空
      • 访问根节点
      • 依次前序遍历树的第 i 棵子树

    • 树的前序 == 二叉树的前序

    • 树的后序 == 二叉树的中序

  • 后序

    • 非空
      • 依次后序遍历树的第 i 棵子树
      • 访问树的根节点

  • 层序遍历

    • 树 —— 编号

    • 森林 —— 子树的根在同一层

  • 求树的高度

    • 空子树 —— 0
    • 非空 —— 森林的高度为 max{子树孙林的高度+1，兄弟森林的高度}

  • 求节点总数

    • 递归前序遍历树

      • 空子树 —— 0
      • 非空 —— == 1 + 左子树森林的结点数 + 右子树森林的结点数

  • 求叶子结点

    • 递归前序遍历树

      • 空子树 —— 子树节点数为0
      • 当前节点无firstChild ，那么也不会有 其他孩子—— ==1 + 兄弟森林中的叶子结点
      • 当前节点有fiestChild —— == 子树森林的叶子结点数 + 兄弟森林的叶子结点

02 习题

  • 树转换成二叉树，结点 u 是结点 v 的父节点的父节点
    • 可能有四种 —— LL、LR、RL、RR —— 父子或者兄弟
  • 叶子结点数目
    • n0 == （k-1)*nk + 1
  • 扫把形的树 —— 非叶子结点，一个一层。
    • 转二叉树 —— 一直只有 firstchild指针非空，到一直只有nextSibiling指针非空
  • 树转二叉树，叶子结点 —— firstchild指针NULL的
  • 知道森林的边和结点数 —— 子树数目
    • 一棵树把所有边都用了，其余的都是只有root结点的树
    • n个结点 —— n-1 条边
  • 二叉树中的右孩子，在树中其实是兄弟关系
  • 二叉树的根节点的右子树，全是森林的树（除了第一棵）
    • 左子树 —— n1 - 1
    • 右子树 —— n2 + n3 + n4
  • F是个森林，B是有F变换的得到的二叉树。若F中有n个非叶子结点
    • B中右指针非空的有多少个
    • 特殊情况推导
      • 森林只有两棵只有根节点的树，转成二叉树 —— 后验证
  • 已知先序和后序，构造森林
    • 二叉树的先序和中序 —— 二叉树
      • 先序——根
      • 中序 + 根 —— 子树
      • 子树 + 先序 —— 根 ——循环
    • 二叉树转森林
      • 加线 —— 根节点 和 除了firstchild 之外的孩子加虚线
      • 删线 —— 所有NR线（实线）删去。
      • 旋转 —— 逆时针旋转 45°
      • 根节点 + 左子树 —— 第一棵树
      • 右子树，每一个RR都是一个根节点
  • 树的后根次序 + 结点的度 ，求树
    • 叶子结点， n0 ， 度数是0
    • B、D、E、F、C、G、J、K、I、L、H、A、
    • 0、0、0、0、3、0、0、0、2、0、2、4
      • DEF—— C（前面度数个结点）
      • JK —— I
      • IL —— H
      • BCGH —— A ( 前面没出现过的 )
