  • 0~5%

01 知识

  • 数组 —— 线性结构的推广

  • 数组 —— 下标和值的偶对的集合，（具有相同类型）

  • C语言，行主序，虚拟内存是一维的。

  • 二维数组中

    • a[ i ] 是第i行的行首地址

  • N维数组可以看做是 元素是 n-1 维数组的向量

    • 向量可以是行向量
    • 也可以是列向量

  • N维数组中元素的个数。

    • π （ 尾下标 - 首下标 + 1 ） == π（每一维度的元素个数）

  • 数组一般不做插入/删除，一般只，存取、修改。

  • 存储方式

    • 行主序
    • 列主序

  • 计算元素地址（存储位置）

    • A[ i ] [ j ] ( c1<i<c2，d1<j<d2 )，每个数据占L个存储单元，首地址 —— LOC[c1] [c2]
    • LOC( i , j ) = LOC[c1] [c2] + L* [ (i-c1) * (d2-c2+1) +j-c2 ]
    • 从 1 开始时再 + 1
      • 基地址 + 偏移地址
      • 偏移地址 —— L * [ 前面有多少行*一行多少个元素 + 本列本元素前有多少个 ] —— L * i，j 之前存了多少
      • 注意 索引/位序 从0开始

  • 压缩存储矩阵（二维数组）

  • 多个值相同的元素分配一个空间，值为0的不分配

  • 两类矩阵

    • 特殊矩阵，非0元素有规律
      • 对称、三角、对角
    • 稀疏矩阵，非零元素少 (<5%)，且无规律
      • 三元组，十字链表

  • 对称矩阵

    • a i j == a j i

    • N^2 个元素 —— n*(n+1)/2 个地址单位中

    • 按行序将 主对角线+下/上三角存储到向量中

    • 对应关系

      • 从0开始（+）

        • 下三角，i > = j

          $$
          \frac{i(i+1)}{2} + j
          $$

        • 上三角，i < j

          $$
          \frac{j(j+1)}{2} + i
          $$

      • 从1开始（-）

        • 分式里面＋变－

  • 三角矩阵

    • 下三角/上三角 元素为常数
    • 对称矩阵多加一个存储单元
      • 最后一个存储单元
        • 0 —— $\frac{n(n+1)}{2}$
        • 1 —— $\frac{n(n+1)}{2} + 1$

  • 对角矩阵

    • 所有非零元素集中在主对角线为中心的带状区域中非零。（奇数条对角线）
    • 三对角矩阵 一共 3n-2 个元素
    • 第一行和最后一行是2个，其余存3个
    • 1 —— 0
      • 主对角线上 i == j
        • k == 3*（i-1）+1 —— 3 i
      • 左下 i == j+1
        • k == 3*（i-1） —— 3 i - 1
      • 右上 i = j-1
        • k == 3*（i-1) + 2 —— 3 i + 1

  • 总公式 —— *k == 2( i - 1 ) + j —— k == 2i + j *

    • k / 3 == i-1 —— i == ( k+1 ) / 3
    • i j 关系由，对角线，右上，左下得到

  • 稀疏矩阵

    • 稀疏因子 —— t / (m*n) <= 0.05

    • 结构

      • 三元组表 triple（row，col，val）数组

        • 矩阵的行、列（大小）、非零元素个数
        • 转置 —— 行列互换，之后按照行序重新排列三元组表
          • 法2：遍历三元组表，找A中列为1的，存到B行为1
          • O(n*len)。

      • 十字链表 TS

        • 行/列 的方向链接所有同一 行/列 的。

      02 习题

    • 常选特值带入验证。

    • 注意下标从什么开始

    • 查找矩阵中书否存在 key。—— O(m+n)

      • 双重循环 —— O(m*n)
      • 矩阵有序
        • 和每一列的最值比较，确定行再遍历行
        • 注意不要越界

    • k == 2 (i-1) + j

    • 三元组，在矩阵中非零元素的个数和位置在操作过程中变化不大时有效

    • 数组是同类型的值和下标偶对的集合
